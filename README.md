## 日本語

AAUI Designer Pro は、ASCII Art（AA）を用いたユーザーインターフェースやマップデザインを、直感的な操作で作成・編集するためのデザイン支援ツールです。TRPGのマップ作成や、レトロな雰囲気のUIデザインに最適です。

### 主な機能
- **直感的なパーツ配置**: ボックス、ボタン、入力欄、テキストなどをグリッドスナップで正確に配置可能。
- **マルチレイヤー管理**: レイヤーの順序入れ替え、不透明度調整、削除保護、個別の配置固定（ロック）に対応。
- **下絵（ガイド画像）機能**: お気に入りの画像を背景に敷いて、トレースするようにAAを作成。Ctrl+ドラッグでのリサイズも可能。
- **高解像度エクスポート**: OSの等幅フォントをスキャンし、アンチエイリアスの効いた超美麗なPNG画像を書き出し。
- **カスタマイズ性**: ショートカットキーの変更、ウィンドウレイアウト（比率）の自動保存。

### インストールと使用方法
1. [Releases](https://github.com/YourUsername/YourRepo/releases) から最新の `Release.zip` をダウンロード。
2. 任意の場所に解凍し、`app.exe` を実行してください。
   - ※ `assets` フォルダは exe と同じ階層に保持してください。
   - ※ 実行時に `configs` フォルダが自動生成され、設定が保存されます。

### 基本操作
- **パーツ追加**: 左パネルの「PARTS」から選択。
- **インライン編集**: パーツをダブルクリックでテキストを直接書き換え。
- **画像リサイズ**: [Ctrl] キーを押しながらガイド画像をドラッグ。
- **配置固定**: [Ctrl + L] または右クリックメニューから、パーツの移動をロック。

## English

AAUI Designer Pro is a specialized design tool for creating and editing ASCII Art (AA) based User Interfaces and maps through intuitive operations. Perfect for TRPG map making and retro-style UI design.

### Key Features
- **Intuitive Part Placement**: Precisely place boxes, buttons, inputs, and text with grid snapping.
- **Multi-Layer Management**: Supports reordering, opacity adjustment, deletion protection, and individual part locking.
- **Guide Image (Tracing)**: Load your favorite images as a background to trace AA. Resizing via Ctrl+Drag supported.
- **High-Res Export**: Scans system monospace fonts to export crystal-clear PNG images with anti-aliasing.
- **Customization**: Keybind editing and persistence of window layout/ratio.

### Installation & Usage
1. Download the latest `Release.zip` from [Releases](https://github.com/YourUsername/YourRepo/releases).
2. Extract to any folder and run `app.exe`.
   - *Note: Ensure the `assets` folder remains in the same directory as the exe.*
   - *Note: The `configs` folder will be auto-generated to save your settings.*

### Basic Controls
- **Add Parts**: Select from the "PARTS" menu in the left panel.
- **Inline Edit**: Double-click any part to edit its label directly.
- **Resize Image**: Hold [Ctrl] and drag the guide image to scale it.
- **Part Lock**: Press [Ctrl + L] or use the right-click menu to fix parts in place.
